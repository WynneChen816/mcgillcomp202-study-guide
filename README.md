# mcgillcomp202-study-guide

## COMP202 / Python Interactive Study Guide

An interactive study guide based on McGill COMP202 (Foundations of Programming), redesigned to help beginners truly understand Python concepts.

---

## ✨ About This Project

Hi! I'm currently studying at McGill and planning to major in Computer Science.

This project started as a personal attempt to actually understand Python, instead of just memorizing things for exams.

While learning, I realized:

- Some concepts are **logically confusing** (like references, aliasing, recursion)
- Some are **memorization-heavy** (like syntax rules, method behaviors)
- Lecture notes are helpful, but not always structured in a way that makes everything *click*

So I rebuilt the course content into something more intuitive.

---

## 🧠 Who This Is For

This site is useful for:

- McGill COMP202 students
- Python beginners
- Anyone who wants clearer explanations of core programming concepts

---

## 🧠 Design Philosophy

This site is designed based on how I personally learn:

- **Logical vs Memory tagging**
  - 🟡 = conceptually tricky (needs deep understanding)
  - 🟢 = memorization-focused

- **Concept-first explanations**
  - not just “what”, but “why”

- **Analogy-based learning**
  - using real-life examples to explain abstract ideas

- **Bilingual thinking (EN + 中文)**
  - helpful if you think in both languages

---

## 🔍 Features

- Organized by lecture order (follows course structure)
- Search function (quickly find topics)
- Language switch (Chinese / English)
- Highlighted key concepts and difficulty types
- Clean structure for fast review
