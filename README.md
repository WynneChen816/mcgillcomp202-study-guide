# mcgillcomp202-study-guide

## COMP202 Interactive Study Guide

An interactive study guide for McGill COMP202 (Foundations of Programming), built from lecture summaries and redesigned based on my own learning process.

---

## ✨ About This Project

Hi! I'm a U0 student in CS at McGill.

This project started as a personal attempt to truly understand COMP202 instead of just “getting through” the course.

While studying, I realized:

- Some concepts are **logically tricky** (like references, aliasing, recursion)
- Some are **memorization-heavy** (like syntax rules, method behavior)
- Lecture notes are helpful, but not always structured in a way that makes everything *click*

So I decided to rebuild the course into something that:

👉 makes concepts clearer  
👉 highlights what actually matters  
👉 matches how *I* think and learn  

---

## 🧠 Design Philosophy

This site is built based on how I personally study:

- **Logical vs Memory tags**
  - 🟡 = need to understand deeply  
  - 🟢 = need to memorize  

- **Concept-first explanations**
  - not just “what”, but “why”

- **Analogy-based learning**
  - using real-life examples to explain abstract ideas

- **Bilingual thinking (EN + 中文)**
  - since many CS students (including me) think in mixed language

---

## 🔍 Features

- Organized by lecture order (follows actual course flow)
- Search function (quickly find topics)
- Language switch (Chinese / English)
- Highlighted key concepts and difficulty types
- Clean visual structure for fast review

---

## ⚠️ About Content

- Most content is based on official lecture notes from COMP202  
- Some parts are **re-explained or emphasized differently** based on what I found difficult while learning  
- This is not an official resource — it’s a **student-built interpretation**
